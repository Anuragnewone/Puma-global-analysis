# PUMA Global Sales Analytics Dashboard | Business Analyst Portfolio Project

## 📌 Project Overview

This project is an end-to-end **Business Analytics** solution developed using **Python, Pandas, NumPy, SQL, Excel, and Power BI**. The objective was to transform raw sales data into interactive dashboards that provide actionable business insights for executives and decision-makers.

The project follows a complete analytics workflow—from raw data preprocessing to business intelligence reporting—demonstrating practical Business Analyst and Data Analyst skills.

## 🔄 Project Workflow

### 1️⃣ Data Collection

* Imported the raw sales dataset (CSV).

### 2️⃣ Data Cleaning (Python)

* Removed duplicate records.
* Handled missing values.
* Corrected inconsistent data.
* Converted data types.
* Standardized categorical values.
* Performed preprocessing using **Pandas** and **NumPy**.
* Exported the cleaned dataset for analysis.

### 3️⃣ SQL Analysis

Solved multiple business questions using SQL, including:

* Revenue Analysis
* Profit Analysis
* Customer Analysis
* Product Analysis
* Store Performance
* Sales Channel Performance
* Return Analysis

### 4️⃣ Power BI Dashboard Development

Built **3 interactive dashboards** using DAX, Power Query, and Power BI.

# 📊 Key Business Insights

* 💰 Total Revenue: **338.07 Million**
* 📈 Total Profit: **182.45 Million**
* 📦 Total Orders: **15,105**
* 🌍 USA generated the highest revenue.
* 🏬 Puma Mumbai was the best-performing store.
* 🏙️ New York generated the highest profit.
* 🛒 Marketplace produced the highest revenue.
* 👕 Track Pants was the best-performing product category.
* 👩 Female customers generated higher revenue than male customers.
* 🔄 Return analysis identified major reasons impacting customer satisfaction.
